# 花呗 AI智伴 · Huabei AI Companion

> 蚂蚁集团产品设计作品 · 作者：邹楠（中山大学大三心理学）

## 项目简介

花呗AI智伴是一款面向手机端的交互式消费管理助手，核心理念是将「理性消费」打造为花呗的差异化定位，通过 AI 重构将用户利益与平台利益从对立关系转化为共生关系。

## 功能模块

| 模块 | 说明 |
|------|------|
| 🏠 首页 | 月度消费总览、AI智伴实时提醒、快捷操作入口 |
| 📊 消费管家 | 分类预算追踪、三级预警开关、AI月度复盘 |
| 💰 分期助手 | 透明分期计算器（总利息/年化利率/还款时间轴）、3天冷静期 |
| ⭐ 成长体系 | Lv.1-4理性等级、每日/周任务积分、等级权益兑换 |
| 👥 理性社区 | 理财群/基金群/附近优惠群，用户帖子点赞互动 |
| 🤖 AI助手 | 语音输入、视频通话（TTS语音播报）、专属人设配置 |

## 技术实现

- **纯原生 HTML/CSS/JS**，无框架依赖，一个文件即可运行
- **AI 对话**：接入 Anthropic Claude API（claude-sonnet-4）
- **语音输入**：Web Speech API（SpeechRecognition）
- **语音播报**：Web Speech API（SpeechSynthesis）
- **视频通话**：WebRTC getUserMedia
- **手机端适配**：375px 移动端布局，模拟真实 App 体验

## 快速体验

直接用浏览器打开 `index.html` 即可，**无需服务器，无需安装依赖**。

> ⚠️ 语音识别和摄像头需要 HTTPS 或 localhost 环境，以及浏览器授权。AI 对话需要网络连接。

## 商业价值

- 逾期率预期降低 20-30%（引导理性消费，降低坏账风险）
- 分期转化率提升 15-20%（信任度提升驱动高质量转化）
- 用户 LTV 提升（延伸至余额宝/理财/保险生态）
- 树立「负责任消费金融」品牌形象，构建差异化竞争壁垒

## 设计理念

> 帮助用户建立理性消费习惯 → 用户逾期率降低 → 平台风险成本降低
> 帮助用户合理使用分期工具 → 用户信任提升 → 分期转化率提升
> 打造差异化产品体验 → 吸引优质用户沉淀 → 用户 LTV 提升
